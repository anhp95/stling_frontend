# Spatiotemporal Linguistics — Frontend

## Running Locally

```bash
npm install --legacy-peer-deps
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

## Environment

Create a `.env` file (or edit the existing one) to point at your backend:

```env
VITE_API_BASE_URL=http://localhost:8000/api/v1
```
